Name: ChatBot Web Application

Language: Python, Javascript, CSS, HTML

Description: End-to-end Fullstack Project ranging from writing backend script for chatbot ai and recommendation system by Python, frontend script comprise of js, css, html to create a Graphical User Interface for the chatbot ai. Applied Flask Framework to create an API hosting on a local server and the use of Ajax - a programming concept for the Web App to send, retrieve data from the server asynchronously in the background without interfering with the display and the behaviour of the website. 

Purpose: A small scale project to demostrate the ability to perform both backend and frontend scripting. And to explore the practical ways of implementation of Flask Framework to make an end-to-end project.
